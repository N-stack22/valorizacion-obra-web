# Estado del arte del proyecto

Este directorio incorpora las tablas del estado del arte elaboradas para el proyecto **Sistema web con enfoque BPM y apoyo de inteligencia artificial para la elaboración del informe mensual de valorización de obra**.

## Archivos incluidos

| Archivo | Ubicación | Uso |
|---|---|---|
| `Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.docx` | `docs/estado-del-arte/` | Versión editable para revisión académica. |
| `Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.pdf` | `docs/estado-del-arte/` | Versión final en PDF. |
| `Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.docx` | `public/docs/estado-del-arte/` | Versión servida desde la aplicación web. |
| `Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.pdf` | `public/docs/estado-del-arte/` | Versión servida desde la aplicación web. |

## Acceso desde la aplicación

En el módulo **Centro de documentos** (`/app/documents`) se añadió una tarjeta llamada **Estado del arte**, con botones para abrir el PDF o descargar el Word editable.

Rutas públicas al desplegar el frontend:

- `/docs/estado-del-arte/Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.pdf`
- `/docs/estado-del-arte/Tablas_Estado_del_Arte_Valorizacion_Obra_Web_50_Articulos.docx`

## Nota de alcance

El documento corresponde al estado del arte del proyecto de valorización de obra. El archivo de ejemplo `Tablas del Estado del Arte.docx.pdf` no forma parte de esta integración porque fue usado únicamente como referencia de formato.
